Early Project • Unity

I Am Ambulance

An endless runner inspired by Subway Surfers, developed as my first team-based game project.

Role

Gameplay Programmer

Focus

Unity, Audio, UI

Engine

Unity / C#

What I Did

This was my first time working on a game in a team environment and collaborating with other disciplines.

I focused mainly on learning Unity workflows and understanding how different parts of a game come together.

My contributions included implementing audio using a singleton-based system and helping with smaller UI features such as credits.

Takeaway

This project was an important starting point for me. It helped me understand team collaboration, game pipelines, and how to work inside an engine for the first time.
